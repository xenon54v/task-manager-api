# Task Manager API

Task Manager API — это backend-приложение для управления задачами.

Проект реализован на FastAPI с использованием SQLAlchemy и SQLite.

## Возможности

- создание задачи;
- получение списка задач;
- получение одной задачи по ID;
- редактирование задачи;
- удаление задачи;
- автоматическая Swagger-документация.

## Стек технологий

- Python
- FastAPI
- SQLAlchemy
- SQLite
- Pydantic
- Uvicorn
